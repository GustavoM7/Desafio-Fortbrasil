# Problema 1:
Implementação de aplicação de gerenciamente de estabelecimentos

### API backend:
Python, pip, Flask

Intalação:

**Ambiente de dev**
### `pip install virtualenv`

### `virtualenv venv`

### `.\venv\Scripts\actvate`

**Resquirementos**

### `pip install -r requirements.txt`


Inicialização:

### `python run.py runserver`

### Frontend:
Web, NodeJs, framework React Js

Instalação:

### `npm i`

Inicialização:

### `npm start`
